## Run
1. Clone this repo
2. Install Node.js(http://nodejs.org) on its page or use Homebrew
3. npm install
4. execute run.sh(linux) or type "node bin/www" in the console
5. Go to http://localhost:3000
6. Please take a look at chrome mobile view mode(F12)
